Car Website

This is a mock website used to learn a few functions from bootstrap. This website uses Dropdown menus, a carousel, panels, and iframes to help users learn about luxury cars in a centeralized location. The carousel automatically changes. The website only links to one page, but this can be easily increased. The linked page also redirects to the main page through a click on the LuxuryCars title.


This is the first page the user sees when navigating to this website. The dropdown menu is shown to show functionality. There are different labels in the dropdown that help seperate cars by type. Panels have filler text. A screenshot of the page that this website will link to. The top bar is the same as the front page, but the LuxuryCar logo links back to the front page. There is an embeded YouTube video and an iframe that links to information about the car.
Carousel is used to cycle through photos so that the user can learn information about new cars available for review on the site. There are included buttons so that the user can click through photos more quickly than the cycle goes. This simply shows how the iframe works in with the original page.
